SlickText Workflow Testing

Testing various flowchart/plumbing/workflow libraries for Slick

Rete.js

  • Has a lot of features - zoom, minimap, good data output.
  • Documentation is confusing and examples (I think) are outdated.
  • Almost too much functionality - there are plenty of things we definitely won't use - like workers

JSPlumb (Community)

  • Super simple - almost DIY.
  • Lack of features - no zoom, no JSON data output.
  • Easy to use.
  • Very customizable.
